## Telepítés

1. Adatbázis (XAMP) elindítása
   
   1.1. Csinálj egy adatbázist `celebrate_together` néven
   
   1.2. Futtasd a `DB.sql` SQL parancsokat rajta 

2. Szerver indítás: ```npm start```
3. Szerver leállítás: *Ctrl+C*
